Read Before Submitting

  1. Prepare all 3 files before starting. You will submit them one at a time in this order: Login Report, then Daywise Summary, then CDR.
  2. Check the date on each file. Make sure all 3 files are for the same date and have not been submitted before.
  3. Do not rename any file. Files must start with the exact names shown below. Renamed files will be rejected.
  4. Wait for confirmation after each upload before moving to the next. Do not close the browser during submission.
  5. Only one person can submit at a time. If the form is locked, wait a few minutes and refresh.
  6. If you see an error after 2 attempts, contact the MIS team immediately.

Required file name prefixes:

• Loginout_Raw_Report • Daywise_Summary_Report • Call_Disposition_Report
